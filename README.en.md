# FlyMall商城

#### Description
FlyMall Rust版，是基于actix-web和rbatis的商城管理系统，一款面向全球的多语言、多业态电子商务平台，系统提供管理、支付、物流、售后等众多应用功能，满足跨境市场的基本需 求，助力更多商户走出国门，大力开拓国外市场，为广大用户提供一站式的国际化电商运营解决方案
